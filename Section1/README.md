<br>

### 주요 개념
1. **데이터 타입**
- Primitive Type: 값이 직접 저장되는 기본 데이터 타입 (int, byte, long 등)
- Reference Type: 주소값을 저장하는 객체 타입 (String, Integer 등)

2. **메모리와 값 전달**
- Call by Value: 값을 복사하여 전달
- Call by Reference: 주소값을 전달하여 참조

3. **시간복잡도**
- O(1): 상수 시간
- O(n): 선형 시간
- O(n²): 제곱 시간
- O(log n): 로그 시간
